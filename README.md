## Synopsis

Enter any text in Ahom script (unicode) and the app will give meaning of all the words at once

## Installation

Install the [requirements](#requirements)
```bash
pip install PySide6
pip install BeautifulSoup
pip install qdarkstyle
pip install requests
```
